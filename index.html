Black-Scholes European Option Pricer

This standalone webpage prices European-style call and put options using the Black-Scholes model. It also reports d₁, d₂, and the main Greeks. You can upload this file directly to a GitHub repository and use GitHub Pages to share it.

Inputs

Choose a European call or put.
Current market price of the underlying asset.
Exercise price written in the option contract.
Example: 0.5 = 6 months, 1 = 1 year.
Enter as a decimal, so 5% = 0.05.
Enter as a decimal, so 20% = 0.20.
Optional. Enter as a decimal, so 2% = 0.02.
Model:
Call: C = S·e-qTN(d₁) - K·e-rTN(d₂)
Put: P = K·e-rTN(-d₂) - S·e-qTN(-d₁)

d₁ = [ln(S/K) + (r - q + σ²/2)T] / [σ√T]
d₂ = d₁ - σ√T

Interpretation notes:
